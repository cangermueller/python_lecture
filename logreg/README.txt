Logistic regression challenge
=============================

1. Open logreg.py
2. Implement the objective function f(theta) (lines 61-76)
3. Implement the derivative fprime(theta) (lines 79-94)
4. Run python logreg.py

If you succeed, the program will terminate, the AUC should be about 0.91,
and you will 'roc.png' will show the ROC curve.
